# Raspberry Piを活用した<br>ハイブリッドセキュアストレージ及び<br>自動運用Bot開発プロジェクト

---

# 背景と目的

**📋 背景・課題**

- クラウドストレージが iPad 上での PDF 書き込みに非対応
- iPad 純正アプリでは同期が遅延・不安定
- 特定デバイス・アプリへの依存を排除したい

**🎯 目的**

- SMB経由でiPadから即座にアクセス可能なローカルNASを構築
- **rclone bisync** によるクラウドとの自動双方向同期（30分毎）を実装
- **Discord Bot** で手動同期・状態監視・操作を実現
- **rsync** による日次・週次の差分バックアップで最大2週間分のデータ保護

---

# システム構成

**【通常時】**

```
クラウドストレージ

    ↑↓ rclone bisync（30分毎 + Bot手動）

ローカル（/mnt/data/cloud）

    ↑↓ Samba（SMB）

iPad（読み書き）
```

**【バックアップ時のみ】**

```
/mnt/data/cloud（クラウドストレージのローカルコピー）

    ↓ rsync（日次・週次 cron + Bot手動）

HDD マウント → 差分バックアップ実行 → アンマウント
（最大2週間の世代管理、普段はアンマウント）
```

# 制約条件・前提条件

**技術的制約**
- クラウドストレージの API 制限あり（同期頻度に上限）
- rclone bisync の競合リスク低減のため同期間隔は 30分毎
- SMB アクセスはローカルネットワーク内限定（外部公開なし）

**前提条件**
- Raspberry Pi 4以上・外付け HDD が利用可能
- クラウドストレージアカウント保有・rclone 連携可能
- Discord サーバー・Bot トークン取得済み
- Linux 基本操作・Python・ネットワーク基礎知識あり
